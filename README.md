# bash-file-organizer
bash file organizer is a bash script that organizes files in a given directory into subdirectories

## what it does
.txt files to /text
.jpg files to /image
files with no extensions to /noextension
.sh files are left in place

## Usage
    bash
bash organise.sh /path/to/target-directory
